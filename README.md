# Solving for LPPL Function Using Different Optimisation Algorithms (2023)

This repository contains the code implementation for this semester thesis project. 

The LPPL function is a model that predicts a market crash from financial data.

 We solve the non-convex optimization problem of fitting the LPPL function to different financial market bubble-crash datasets using various optimization methods - Genetic Algorithms (GA), Particle Swarm Optimization (PSO), as well as a combination of different methods e.g. GA first -> PSO second. 

Our goal is to investigate which optimization method performs best. 

See the file OptEng_Report.pdf for the most in-depth explanation of the project.
